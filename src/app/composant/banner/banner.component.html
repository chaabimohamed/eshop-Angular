About Us

Ce projet s'inscrit dans le cadre des projets e-commerce que la société VevioSys a créée en réel. L'objectif de mon travail consiste à développer une application web de commerce. La partie front sera réalisée en utilisant les techniques web modernes qui base sur les "Component'" ANGULAR. La partie Back sera constituée d'un REST API un langage JAVA est le Framework spring. Pour la gestion des tâches on utilise des outils comme SCRUM, GIT, GITHUB.
